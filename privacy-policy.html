1. Introduction

Welcome to LIKApps ("we," "our," "us"). We are committed to protecting your privacy and ensuring transparency about how we collect, use, and safeguard your personal information across all our web applications. This comprehensive Privacy Policy explains our practices regarding the collection, use, and disclosure of information through our platform and all web applications developed under the LIKApps ecosystem.

This policy applies to multiple distinct web applications including LIKApos, LIKAshare, LIKAnote, LIKAsys, LIKAdev, and any future applications developed under the LIKApps platform. Each application, while having unique functionalities and purposes, adheres to the same privacy standards and data protection practices outlined in this policy.

2. Definitions

For the purposes of this Privacy Policy:

  • "LIKApps Platform" refers to the entire ecosystem of web applications developed and maintained by LIKApps, including LIKApos, LIKAshare, LIKAnote, LIKAsys, LIKAdev, and any future applications.
  • "Individual Applications" refers to each distinct web application (LIKApos, LIKAshare, LIKAnote, LIKAsys, LIKAdev) that operates under the LIKApps platform.
  • "Personal Information" means any information relating to an identified or identifiable natural person.
  • "Services" refers collectively to all web applications, websites, and services provided under the LIKApps brand.
  • "User" means any individual who accesses or uses any LIKApps application.
  • "Google OAuth 2.0" refers to the authentication protocol provided by Google LLC that allows users to log in to our applications using their Google account credentials.

4. Information Collection

4.1 Application-Specific Data Collection

Each LIKApps application may collect information specific to its functionality:

  • LIKApos: Transaction data, inventory information, customer records, payment processing data
  • LIKAshare: File metadata, sharing permissions, collaboration data, access logs
  • LIKAnote: Note content, organizational data, user preferences, synchronization data
  • LIKAsys: System configuration, performance metrics, administrative settings
  • LIKAdev: Code repositories, development projects, debugging information, deployment data

4.2 Common Data Collection Across All Apps

All LIKApps applications collect the following common information:

  • Account registration and profile information
  • Authentication data (via Google OAuth 2.0)
  • Usage statistics and application performance data
  • Technical information (browser type, device information, IP address)
  • Communication and support data
